¿Como instalar Ángular?

Ángular es un framework de JavaScript, también se usa con TypeScript.

  1. Node.js

    2. Se descarga la verión LTS que tiene un soporte a largo plazo desde:

    Sitio Oficial
  2. Atención:

    3. Ángular se instala por consola!

  3. Instalación:

    4. Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned

    se coloca esto por si hay problemas con el power shell

    npm install -g @angular/cli
  4. Verificamos
    5. npm -v

¿Como crear componentes?

En la terminal de visual Studio podemos agregar los componentes de la siguiente manera.

    ng g c componentes/navbar

    Crea una carpeta llamda componentes y dentro de ella se encuentra el componente navbar

¡Comandos básicos que se usan en ángular!

  • ng new nombreProyecto
  • ng version
  • ng s --0
  • ng g c nombreCarpeta/componente

¿Que es Ángular?

Angular es un framework de ingeniería de software de código abierto mantenido por Google, que sirve para desarrollar aplicaciones web de estilo Single Page Application (SPA) y Progressive Web App (PWA). Sirve tanto para versiones móviles como de escritorio.
