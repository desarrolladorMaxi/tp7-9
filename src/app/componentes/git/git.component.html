Subir archivos a GitHub

Te posicionas en la carpeta que creaste para alojar tu página.

  • git init
  • prepara el repositorio para ser subido
  • git .\nombre.html\
  • si es un solo archivo a subir
  • git add .
  • si vas a subir todos los archivos
  • git commit -m "primer commit"
  • breve descripcion de la mejora
  • git branch -M main
  • subis la rama
  • git remote(...)
  • pones la dirección dónde lo vas a subir
  • git push -U origin main
  • lo subis
