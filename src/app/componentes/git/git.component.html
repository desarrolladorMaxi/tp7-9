1. git init
    Prepara el repositorio para ser subido
  2. git .\nombre.html\
    Si es un solo archivo a subir
  3. git add .
    Si vas a subir todos los archivos
  4. git commit -m "Primer Commit"
    Breve descripcion de la mejora
  5. git branch -M main
    Subis la rama
  6. git remote(...)
    Pones la dirección dónde lo vas a subír
  7. git push -U origin main
    Lo subís
